How to download Github repository using terminal

B1: mở đến trang Github: https://github.com/ChantalMB/MacOdrum-Coding-for-DH.git

 B2: gõ git clone và paste đường link trên vào terminal

 Ex:  
```
(base) PS C:\MINE\CARLETON\Courses\coding_for_humanities\wsfall> cd ..
(base) PS C:\MINE\CARLETON\Courses\coding_for_humanities> git clone https://github.com/ChantalMB/MacOdrum-Coding-for-DH.git
```
